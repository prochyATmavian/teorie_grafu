🚀 JAK ZAČÍT:

LINUX/MAC:
  1. bash tools/setup_venv.sh
  2. ./analyze.sh

WINDOWS:
  1. python -m venv venv
  2. venv\Scripts\activate
  3. pip install -r requirements.txt
  4. deactivate
  5. analyze.bat
