🚀 RYCHLÝ START:

  LINUX / MAC:
    1. bash tools/setup_venv.sh
    2. ./analyze.sh
       → Vyberte: 1) Vlastnosti, 2) Matice, 3) Kompletní

  WINDOWS:
    1. python -m venv venv
    2. venv\Scripts\activate
    3. pip install -r requirements.txt
    4. deactivate
    5. analyze.bat
       → Vyberte: 1) Vlastnosti, 2) Matice, 3) Kompletní
